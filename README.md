# My Personal Website
[https://leexiao.site/](https://leexiao.site/)

I built this site to help myself learn efficiently and realise some of my ideas.



## Install
node version 18.20.2
pnpm 9.1.0

```shell
pnpm install
```

## Run

```shell
pnpm dev
```

## Test

```shell
pnpm test
```



## Build

```shell

release by github actions

```
